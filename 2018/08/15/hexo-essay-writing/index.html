Hexo Essay Writing

2018-08-15
  • skills
  1. 打开git,进入MyBlog目录；
  2. 输入 hexo new "essay-name"；
  3. 打开source文件夹，找到 essay-name.md，使用markdown语法写作；
  4. 执行命令 / hexo clean / hexo generate/ hexo deploy；
  5. over；

假如hexo命令出现问题，基本上就是_config.yml 文件的空格有问题，看报错就好了。
